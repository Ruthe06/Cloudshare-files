Offline File Share (ShareMe Style)

Peer-to-peer, pure offline file transfer via browser.
Both devices must be connected to the same Wi-Fi or hotspot, no internet required.
How it works:
  1. Sender: Open "Offline Sender", select file, create offer and share it with receiver.
  2. Receiver: Open "Offline Receiver", paste offer, make answer, and return answer to sender.
  3. Once connected, file transfer will start directly device-to-device!
